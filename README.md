# 快速文件访问
***
想必大家都遇到过文件文件目录太深，又时常需要打开相关文件的尴尬场景。


我个人因为也遇到这样的问题所以想写一个提高工作效率的小工具。

代码已经开源欢迎大家查看，或者添加新的功能。

***
开发环境VS2017+WPF。

***
想要设置开机启动可以把快捷方式放到
> C:\Users\（用户名）\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

文件夹下面这样程序就会在开机时自启动。
